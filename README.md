## Description

Ludens is a lightweight PHP framework designed to be the as elegant and modern as possible. It provides a clean foundation for building applications that value simplicity, clarity, and control.

## Features (November 2025)

- Modular Core - small, composable components (Routing, HTTP, View, etc...).
- Custom Application Kernel with configuration, caching and environment loading.
- Routing System with method chaining.
- Twig based View Renderer.
- Simple Request/Response Abstraction.
- PDO-based Database Layer.
- Session Flash System for validation and notifications.
